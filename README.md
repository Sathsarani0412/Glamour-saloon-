# Glamour Salon Website

## Overview

Glamour Salon Website is a responsive web application designed to provide customers with an easy and convenient way to explore salon services and book appointments online. The website features an attractive user interface and supports customer interaction through appointment booking and contact forms.

## Features

* Home page with salon introduction and featured services
* About page with salon information
* Gallery page showcasing salon work and styles
* Contact page for customer inquiries
* Online appointment booking system
* Responsive design for desktop and mobile devices
* SQLite database integration for storing appointment and contact information

## Technologies Used

* HTML5
* CSS3
* JavaScript
* SQLite

## Functionalities

### Appointment Booking

Customers can submit appointment requests by providing their details, preferred service, date, and time. The information is stored securely in the SQLite database.

### Contact Management

Customers can send inquiries through the contact form, and the submitted information is stored in the database for review.

### Gallery Display

The gallery page showcases salon services, hairstyles, and beauty treatments through an organized image collection.

## Project Purpose

This project was developed to demonstrate full-stack web development concepts, including front-end design, client-side interactivity, database integration, and appointment management functionality.
